# WikiFX
A lightweight file base wiki application. 

## This wiki system will supports:
* Markdown
* Latex formula
* Html Rendering
* Docx Export

## Planning Features 
- [ ] Pdf Export
- [ ] Git Revisions for text pages
- [ ] Asciidoc Integration

## Using APIs:
* Spring Framework
* JFxtras
* Docx4J
* ControlsFX
* FontAwesomeFX
* Eclipse Modeling Framework
* JGit
